Hi, I'm Bo. |


Hey, I am Bo Huang, a self-taught web developer. In the past two years I went to Centennial College for Electronics Engineering. In the second year of college, I took a System Integration course where we were assigned a task of building a project using HTML and JavaScript for front end, using Python and MySQL for back end. The moment when I clicked on a button and had some data fetched from database and displayed on screen, I was completely amazed. That was when I found what I am really passionate about. I want to be a web developer.

After graduating I joined freeCodeCamp to learn more up-to-date technologies. Over the past year I have completed 15 frontend projects using HTML5, CSS3(Sass), Bootstrap4, JQuery, React, Redux and D3. Recently, I have been building backend projects using Node.js(Express.js) and MongoDB(Mongoose). To see my recent work, please go to the Work section below.
